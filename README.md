# OFM Typing Cheat Extension

A Brave/Chrome extension to automate the typing test on `ofmjobs.com`.

## Features
- **In-page Integration**: Adds a "Start Cheat" button directly to the typing test.
- **Customizable Speed**: Set your target WPM via the extension popup.
- **Human-like Simulation**: Random jitter between keystrokes to mimic real typing.
- **Works on Brave**: Tested and optimized for the Brave browser.

## Installation Instructions

1. **Download the Extension**:
   - You can download the pre-packaged [OFM_Typing_Cheat.zip](https://github.com/tryMighty/typing-test-cheat/blob/master/OFM_Typing_Cheat.zip) directly from this repository and extract it.
2. **Open Extensions Page**:
   - In Brave/Chrome, go to `brave://extensions/` or `chrome://extensions/`.
3. **Enable Developer Mode**:
   - Toggle the **Developer mode** switch in the top right corner.
4. **Load the Extension**:
   - Click **Load unpacked**.
   - Select the folder where you extracted the files (the folder containing `manifest.json`).
5. **Verify**:
   - "OFM Typing Cheat" should now appear in your extensions list.

## How to Use

1. **Set your WPM**:
   - Click the extension icon in your toolbar.
   - Enter your desired WPM (default 60).
   - Click **Save Config**.
2. **Take the Test**:
   - Go to [ofmjobs.com/tests/typing](https://ofmjobs.com/tests/typing).
3. **Start the Cheat**:
   - Look for the green **Start Cheat** button in the bottom-right corner.
   - Click it and watch the magic happen!
